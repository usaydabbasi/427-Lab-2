# GGR472 LAB 2: Introducing Mapbox GL JS 

This repository contains files for GGR472 Lab 2, which demonstrates how GeoJSON data can be visualized in a web map using Mapbox GL JS.

The webpage displays public libraries and parks across Mississauga, Ontario.

## Webpage
https://usaydabbasi.github.io/427-Lab-2/

## Key Repository Contents

- **index.html**  
  HTML file that creates the webpage and loads the Mapbox map.

- **style.css**  
  CSS file used to style the webpage layout and header.

- **script.js**  
  JavaScript file that initializes the Mapbox map and adds GeoJSON layers.

- **libraries.geojson**  
  GeoJSON file containing library locations.

- **parks.geojson**  
  GeoJSON file containing park locations.
